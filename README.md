# 🎆cxl-vue-template
🎆 基于Vue3+elementUI-Plus+TypeScript+Axios+MetroUI自用快速开发vue项目的模板


# 安装使用


```bash
git clone https://github.com/18696232390/cxl-vue-template.git
cd cxl-vue-template
npm install
npm run serve
```

访问 http://127.0.0.1:8080/ 即可

默认登录账户密码 admin/123456



# 功能介绍
- 集成 ElementUI-Plus ✔️
- 集成 MetroUI ✔️
- 集成 axios ✔️
- 集成 sql-formatter ✔️
